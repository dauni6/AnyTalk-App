# AnyTalk-App(share every day life. communicate app)


트위터 클론 형식으로 만들어본 '애니톡'앱 입니다.

구글 플레이 URI : https://play.google.com/store/apps/details?id=com.twitter.twitterclone


1. Functions
    * 회원가입, 탈퇴, 비밀번호 변경
    * 새로운 피드(글) 작성, 사진,GIF 첨부가능
    * 해시태그 검색가능
    * 좋아요
    * 유저 팔로우
 
 
2. Server
    * Firebase
 
3. Dependencies
    * Firebase Database
    * Firebase Storage
    * Firebase Auth
    * Firebase Core
    * Multidex
    * Glide
